---
description: List and manage macOS Reminders
allowed-tools: Bash
---

Run `mac-reminders` to show pending reminders.

Available commands:
- `mac-reminders` or `mac-reminders list` - pending reminders
- `mac-reminders completed` - completed reminders
- `mac-reminders all` - all reminders
- `mac-reminders --json` - JSON output

Present the reminders in a clean, readable format.
